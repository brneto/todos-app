Todo App Demo
======
A small PWA React/Redux application using known best practices, design pattern and libraries to achieve a sofware highly scalable.

Usage
------
Download modules:

	$ yarn

Run app server in development:

	$ yarn start

Stop app server:

	$ yarn stop

Build
------
	$ yarn build

Generated files will be in the `dist` folder.
